  • This website was designed specifically for the Software Engineering unit 2 written assignment.

    The purpose of this page is to show basic information about Bloomstar Bouquet, a floral company based in Canada. All the information and pictures are fictitious and do not represent official business facts.

  • Simple website version

    According to the unit 1 case study, Bloomstar was facing some challenges to keep its physical stores open, so the new CEO decided to invest in a new online store to sell its flowers. This page can be considered the actual old-fashioned website that needs to be completely changed to become the wondered e-commerce.

  • Written assignment's main objective

    Taking into consideration that the reason for building this website is to get familiar with the GitHub Pages tool, the author of this page did not create a fully functional online store, even because I am not a web developer. So my lack of front-end and back-end knowledge is a barrier to developing practical e-commerce.
